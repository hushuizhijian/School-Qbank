2. 画圆时, 圆规针尖固定的一点是圆的( ), 它决定圆的( ), 圆规的两脚叉开的距离是所画圆的( ), 它决定圆的( )。  
3. 一个车轮转动一周,前进多少米是指圆的( )。  
4. 把一张圆形纸片至少折( )次, 就可以找到圆心。
