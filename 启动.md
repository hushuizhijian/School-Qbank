# 小学数学智能题库系统 V2 — 启动指南

## 环境要求

- Python >= 3.10
- Node.js >= 18
- Windows / macOS / Linux

## 首次启动 — 安装依赖

### 后端

```powershell
cd F:\tools4\backend
pip install fastapi "uvicorn[standard]>=0.30.0" "sqlalchemy[asyncio]>=2.0.0" aiosqlite "pydantic>=2.0.0" "pydantic-settings>=2.0.0" python-multipart PyMuPDF Pillow httpx "python-jose[cryptography]>=3.3.0" "passlib[bcrypt]>=1.7.4" celery redis onnxruntime opencv-python-headless numpy
```

### MinerU SDK（项目自带，PDF解析引擎）

```powershell
pip install "F:\tools4\MinerU-Ecosystem-main\sdk\python"
```

### 前端

```powershell
cd F:\tools4\frontend
npm install
```

## 启动服务

### 1. 启动后端（先开一个终端）

```powershell
cd F:\tools4\backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### 2. 启动前端（再开一个终端）

```powershell
cd F:\tools4\frontend
npm run dev
```

## 配置说明

后端配置在 `backend/.env` 文件中，关键配置：

```
DATABASE_URL=sqlite+aiosqlite:///./data/schoolwork.db
DEEPSEEK_API_KEY=你的DeepSeek API Key
DEEPSEEK_BASE_URL=https://api.deepseek.com
```

## MinerU Token 配置（PDF解析）

MinerU 云端解析需要 Token，按以下优先级读取：
1. 环境变量 `MINERU_TOKEN`
2. `backend/.env` 中的 `mineru_token`
3. `backend/token.txt` 文件

## 注意事项

1. 首次启动后端会自动创建数据库和预设知识点数据
2. 前端开发服务器端口可能自动切换到 5174（当 5173 被占用时）
3. 上传的 PDF 文件存储在 `backend/data/uploads/` 目录
4. 解析生成的图片存储在 `backend/data/images/` 目录
5. Redis 为可选依赖（仅 Celery 异步任务需要），不启动 Redis 不影响核心功能
6. MinerU SDK 位于 `MinerU-Ecosystem-main/sdk/python/`，需手动安装
